# 📊 Feedback Analyzer

> A lightweight Python application that performs sentiment analysis on customer feedback using keyword-based classification.

---

## 🚀 Overview

The **Feedback Analyzer** is a command-line Python tool that classifies customer feedback into three categories:
- Positive 😊  
- Negative 😡  
- Neutral 😐  

It uses simple Natural Language Processing (NLP) techniques based on keyword matching to analyze input text.

---

## ✨ Features

- Real-time feedback analysis
- Sentiment classification (Positive / Negative / Neutral)
- Text preprocessing (case normalization & punctuation handling)
- Simple and clean command-line interface
- Beginner-friendly Python implementation

---

## 🧠 How It Works

1. User inputs a feedback sentence  
2. Text is cleaned (lowercase + punctuation removal)  
3. Words are compared against sentiment dictionaries  
4. A sentiment score is calculated  
5. Final classification is displayed  

---

## 🛠️ Tech Stack

- Python 3
- Basic NLP (keyword-based analysis)
